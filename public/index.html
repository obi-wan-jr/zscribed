Get All Books

GET
/api/books
Retrieve a list of all 66 books in the Bible with their chapter counts.
curl http://localhost:3005/api/books
{ "success": true, "data": [ { "id": "GEN", "name": "Genesis", "chapters": 50 }, { "id": "EXO", "name": "Exodus", "chapters": 40 } ] }

Get Specific Book

GET
/api/books/{bookId}
Get information about a specific book by its ID.

Path Parameters

bookId string
curl http://localhost:3005/api/books/GEN
{ "success": true, "data": { "id": "GEN", "name": "Genesis", "chapters": 50 } }

Get Chapter

GET
/api/books/{bookId}/chapters/{chapter}
Retrieve all verses from a specific chapter.

Path Parameters

bookId string
chapter number
curl http://localhost:3005/api/books/GEN/chapters/1
{ "success": true, "data": { "reference": "Genesis 1", "verses": [ { "book_id": "GEN", "book_name": "Genesis", "chapter": 1, "verse": 1, "text": "In the beginning God created the heavens and the earth." } ] } }

Get Verse Range

GET
/api/books/{bookId}/chapters/{chapter}/verses?start={start}&end={end}
Get a specific range of verses from a chapter.

Path Parameters

bookId string
chapter number

Query Parameters

start number (optional)
end number (optional)
curl "http://localhost:3005/api/books/GEN/chapters/1/verses?start=1&end=5"
{ "success": true, "data": { "reference": "Genesis 1", "verses": [ { "book_id": "GEN", "book_name": "Genesis", "chapter": 1, "verse": 1, "text": "In the beginning God created the heavens and the earth." } ] } }

Search Verses

GET
/api/search?q={query}&book={bookId}&chapter={chapter}
Search for verses containing specific text.

Query Parameters

q string (required, min 3 chars)
book string (optional)
chapter number (optional)
curl "http://localhost:3005/api/search?q=love&book=1JN"
{ "success": true, "data": { "query": "love", "results": [ { "book": "1JN", "bookName": "1 John", "chapter": 4, "verses": [ { "book_id": "1JN", "book_name": "1 John", "chapter": 4, "verse": 8, "text": "Whoever doesn't love doesn't know God, for God is love." } ] } ] } }
